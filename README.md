# Echoes of the Past

A Cultural Heritage Digital Archive of India designed to preserve and showcase historical artifacts through digital technology.

## Features

* Digital Archive of Heritage Artifacts
* Search and Category Filtering
* Interactive 3D Artifact Visualization
* 360° Virtual Heritage Tours
* Firebase Firestore Integration
* Admin Content Management System
* AI-Powered Heritage Guide

## Technologies Used

* HTML5
* JavaScript
* Firebase Hosting
* Firebase Firestore
* Three.js
* Gemini API
